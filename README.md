# Travel Science Website

A static product website for Travel Science, by Kelvcoop.

## Local Preview

```bash
npm run dev
```

Then open:

```text
http://localhost:4173/
```

## Project Structure

```text
src/
  components/
  data/
  pages/
  utils/
public/
  assets/
```

Runtime assets referenced as `/assets/...` are served from `public/assets`.
